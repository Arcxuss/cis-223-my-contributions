Variables

  • Declare - set a location in memory to store data
  • Assign - set value stored in variable
  • Initialize - declare a variable and assign a value

Keywords

  • let - block scoped variable
  • var - function scoped variable (old)
  • const - block scoped constant (cannot assign a new value after initialization)

Relative Assignment Operators

  • ++  increment
  • --  decrement
  • +=  addition assignment
  • -=  subtraction assignment
  • *=  multiplication assignment
  • /=  division assignment
